## Problem

Sports coaches, players, and parents needed a unified platform for performance evaluation, real-time feedback, and transparent communication around player development.

## Approach

Built the iOS app in Swift with real-time capabilities via WebRTC. Designed game card generation, performance analytics, and self-evaluation tracking workflows for coaches and players alike.

## Key Features

- Game card generation with real-time feedback
- Performance analytics for players
- Self-evaluation tracking
- Parent portal for oversight
- Motivational training modules

## Result

Published on the App Store, serving coaches and sports organizations with a comprehensive evaluation and communication tool.
