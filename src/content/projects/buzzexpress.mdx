## Problem

Users needed an accessible mobile tool to create professional-looking posters and greeting cards without design expertise or desktop software.

## Approach

Built a full-featured Android app in Kotlin with a customizable template library. Implemented a layered editing system supporting text, stickers, overlays, filters, and effects. Used Room for local data persistence and Retrofit for template fetching.

## Key Features

- Customizable template library
- Text, sticker, and overlay editing
- Filters, effects, and adjustments
- Collage and montage creation
- Drawing and painting tools
- Frames, borders, resize, and reshape

## Result

Published on Google Play Store, serving users who need quick, professional poster and greeting card creation on mobile.
